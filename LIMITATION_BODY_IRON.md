# Критичное ограничение: Body Iron target применим только для женщин репродуктивного возраста

## Проблема

NHANES собирает **sTfR (soluble transferrin receptor)** только для **женщин репродуктивного возраста (12-49 лет)**.

**Данные из валидации:**
- Всего женщин с sTfR: **6303**
- Возрастной диапазон: **12-49 лет** (средний 29.4, медиана 29)
- **0% данных для женщин ≥50 лет**
- **0% данных для мужчин**

**RIDEXPRG (статус беременности):**
- Собирается для женщин 8-59 лет
- Публикуется только для 20-44 лет (disclosure risk) → **39.9% NaN в наших данных**
- Из тех, у кого RIDEXPRG доступен:
  - 56.1% не беременны (подтверждённо отрицательный тест)
  - 2.7% беременны
  - 1.3% статус не определён

**Вывод:** sTfR собирается не только для беременных, но **строго для женщин репродуктивного возраста**.

---

## Последствия для модели

### 1. Body Iron target (таргет Y_IRON_DEFICIENCY, Y_IDA)

**Формула Body Iron:**
```
Body Iron = -(log10(sTfR / ferritin) - 2.8229) / 0.1207
```

**Применимость:**
- ✅ Женщины 12-49 лет: **валидирована** (N=6303)
- ❌ Женщины ≥50 лет: **НЕ валидирована** (N=0)
- ❌ Мужчины (все возраста): **НЕ валидирована** (N=0)

**Ограничение**: ~40-50% потенциального рынка (мужчины + женщины >50) **исключены из scope**.

---

### 2. Alternative target: Ferritin-only (без sTfR)

Для полного охвата можно обучить альтернативную модель на **ферритин <30 ng/mL** (без Body Iron):

**Плюсы:**
- Применима для мужчин и женщин >50 лет
- Ферритин собирается для всех групп в NHANES

**Минусы:**
- Ферритин зависит от воспаления (CRP) — менее точный таргет
- Более высокий false positive rate при инфекциях/воспалении
- Потребует CRP как рефлекс-тест для уточнения

**Trade-off**: точность Body Iron (женщины 18-49) vs. охват ферритина (все группы, но с оговорками).

## Стратегия для пилота

### Вариант A: Честное ограничение (рекомендовано для старта)

**Таргет**: Body Iron (Y_IRON_DEFICIENCY)

**Применимость**: Женщины 18-50 лет

**Позиционирование:**
- КДЛ: "Умный триаж для женщин репродуктивного возраста"
- 29н B2B: "Скрининг дефицита железа у сотрудниц"
- B2C: "Оценка железа для женщин до 50 лет"

**Disclaimers:**
- "Модель обучена на данных женщин 18-49 лет"
- "Для мужчин и женщин >50: рекомендуем стандартный протокол (ферритин + при необходимости CRP)"

**Преимущества:**
- Высокая точность (Body Iron — gold standard)
- Независимость от воспаления
- Честность = доверие

**Недостатки:**
- Ограниченный TAM (~40-50% рынка)

---

### Вариант B: Две модели (для масштаба)

**Модель 1: Body Iron** (женщины 18-49)
- Таргет: Y_IRON_DEFICIENCY (Body Iron < 0)
- Фичи: ОАК + демография + биохимия
- AUROC ожидаемый: ~0.82

**Модель 2: Ferritin-adjusted** (мужчины + женщины ≥50)
- Таргет: Ferritin <30 ng/mL (из NHANES, где есть данные для всех)
- Фичи: ОАК + демография + биохимия + **CRP** (для корректировки)
- AUROC ожидаемый: ~0.75-0.78 (ниже из-за воспаления)

**Маршрутизация:**
```python
if (sex == 'F') and (age >= 18) and (age <= 50):
    model = model_body_iron
    output = "Body Iron risk (gold standard)"
else:
    model = model_ferritin
    output = "Ferritin risk (adjust for CRP if elevated)"
```

**Преимущества:**
- Полный охват рынка (100%)
- Оптимальная точность для каждой группы

**Недостатки:**
- Сложнее в разработке (две модели, два pipeline)
- Модель 2 требует CRP (дополнительные расходы)

## Рекомендация

**Для MVP (3-6 месяцев):**

→ **Вариант A** (Body Iron, женщины 18-50)

**Причины:**
1. Простота (одна модель)
2. Высокая точность (gold standard)
3. Достаточный рынок для proof-of-concept:
   - B2B корпоративный: женщины 18-50 — ~40-50% сотрудников
   - КДЛ: можно таргетировать женщин репродуктивного возраста (высокий risk group для ЖДА)
4. Честность = доверие (критично для медицинского AI)

**Для Scale (после пилота):**

→ **Вариант B** (две модели)

**Этапы:**
1. Собрать российские данные с sTfR для мужчин и женщин >50 (или договориться с КДЛ на retrain)
2. Обучить Модель 2 на ферритине (для тех, кто вне scope Модели 1)
3. Интегрировать маршрутизацию
